## Boardroom Agent – A Multi-Agent Meeting Intelligence System using LangGraph

**Boardroom Agent** is an AI-powered multi-agent meeting intelligence system that allows users to generate **meeting transcripts, summaries, and insights** simply by pasting a meeting link.

An intelligent agent automatically joins the meeting, processes the conversation, and generates:
- Accurate meeting transcription  
- Concise summaries  
- Key discussion points and insights  

The system is designed as a **multilingual application**, making it suitable for **multiple Indian languages**, enabling broader accessibility across diverse users.

This project demonstrates the use of **Agentic AI architecture** for automating meeting understanding and documentation workflows.
